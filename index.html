About
The person behind the pipeline

I joined Fiera Real Estate UK in 2022 as a graduate hire and quickly became the firm's central data person. What started as Excel and basic reporting has grown into a full data function spanning Airtable, Power BI, SQL Server, and Python, all built and maintained independently.

I work across the entire business: investment teams rely on my dashboards and deal pipelines, investor relations use the platforms I've built for meeting tracking and reporting, and finance teams use automated data flows I've connected to their accounting systems.

More recently I've taken on the firm's AI agenda, running internal workshops, building AI-powered data extraction tools, and acting as the bridge between emerging technology and practical business use. I care about making data useful, not just available.

Tools I've built
Systems that run the business

These aren't side projects. They're production tools used daily by investment professionals, portfolio managers, and senior leadership.

DigiDebt

An automated deal pipeline for the debt investment team. Broker opportunities hit an email trigger, get ingested into Airtable via automation, and AI fields extract 20+ structured data points from unstructured deal summaries. Two Power BI dashboards sit on top for portfolio monitoring and mandate tracking.

Airtable Power BI AI fields Automation

Investor Intelligence Hub

A three-phase Airtable platform replacing fragmented Salesforce and Excel workflows for the IR team. Centralises investor meeting tracking, contact management, and reporting into a single source of truth. Approved at board level.

Airtable Salesforce API IR workflow

PBSA Investment Heatmap

A Python tool for the investment team that overlays HESA university enrolment data with TravelTime API isochrones to visualise student housing catchment areas. Used to identify undersupplied markets and inform acquisition strategy.

Python HESA data TravelTime API Geospatial
Projects
Selected work

Deeper dives into problems I've solved and systems I've shipped.

Xero Accounting Pipeline

Built a Python-to-SQL Server pipeline pulling live data from Xero's Accounting API into a three-table schema (accounts, trial balance, journals), feeding Power BI dashboards for the finance team. Replaced manual spreadsheet reconciliation.

Python SQL Server Xero API Power BI

Valuation Report Data Extraction

An Airtable workflow using AI fields to parse commercial property valuation PDFs, extracting NIY, equivalent yield, market value, and valuation dates. Handles large multi-property reports with structured output ready for reporting.

Airtable AI extraction PDF parsing

Asset Image Pipeline

A GitHub Actions pipeline that syncs property images from Airtable to GitHub, replacing expiring attachment URLs with permanent links across three Airtable bases. Ensures dashboards and reports always display current imagery.

GitHub Actions Python Airtable API
